VirtualBox "Appliance is not signed" Error Message

If you see the following Appliance is not signed error message.

It can be safely ignored. [1]

Use provided OpenPGP signatures instead, see: Verify Kicksecure ™ Images.

Forum Discussion[edit]

  • Appliance is not signed

Footnotes[edit]

  1. ↑ Kicksecure ™ does not use this signing mechanism and nobody else is known to use it either. VirtualBox provides neither documentation on how to use this feature nor a tool to use this feature. The provided OpenPGP signatures are a more complete and secure replacement of this feature.
