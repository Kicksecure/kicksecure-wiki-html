Installation[edit]

One option might be to install using flatpak from flathub. Special instructions are also required in this case.

1. Installation of flatpak from Debian.

Install flatpak.

1. Update the package lists. 

sudo apt update

2. Upgrade the system. 

sudo apt full-upgrade

3. Install the flatpak package.

Using apt command line parameter --no-install-recommends is in most cases optional. 

sudo apt install --no-install-recommends flatpak

The procedure of installing flatpak is complete.

2. Add flathub repository. [1]

  • Non-Qubes: 
    flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

  • Qubes TemplateVM: 
    http_proxy=http://127.0.0.1:8082 flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

3. Install Chromium from flathub repository. [2] [3]

  • Non-Qubes: 
    flatpak install flathub org.chromium.Chromium

  • Qubes TemplateVM: 
    http_proxy=http://127.0.0.1:8082 flatpak install flathub org.chromium.Chromium

Usage[edit]

flatpak run org.chromium.Chromium

[4]

Update[edit]

  • Kicksecure ™: 
    flatpak update

  • Kicksecure ™ for Qubes TemplateVM: 
    http_proxy=http://127.0.0.1:8082 flatpak update
